My Projects

SMART SHOE

Smart Shoe for the Blind: Developed a wearable device using ultrasonic sensors, Arduino board and a buzzer, The sensor is programmed with a reasonable range between obstacle, it sends signal to the buzzer whenever there is obstacle ahead, so the buzzer will beep very hard to alert the wear that there is a danger. This makes the wearer channel is way from approaching danger. This is to help visually impaired individuals navigate obstacles and enhanced mobility and safety.

Smart Dustbin

Smart Dustbin: Created an automated waste management system that uses ultrasonic sensor, servo motor, Arduino board, jumper wires and the dustbin itself. This smart dustbin detects trash and send signal to the servo motor to make it open for successive jumping of the trash. Also it alerts sanitation workers when bins need to be emptied, optimizing collection efficiency.
